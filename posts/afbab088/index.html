cisco路由器配置

发表于2020-10-22|更新于2020-10-22|计算机网络

路由器enable开始配置，config terminal进入全局模式
shutdown关闭接口 no shutdown打开接口 ip address配置接口ip地址和子网掩码
hub和switch是同类设备，要用交叉线连接
路由器的每个广域网接口也需要配置IP地址和子网掩码
show ip route显示连接的所有网段
ip route配置路由表(两边的路由都要配置)
interface xxx配置接口
Router#config te
Router#config terminal
Enter configuration commands, one per line. End with CNTL/Z.
Router(config)#ip route 12.0.0.0 255.255.0.0 11.0.0.2
Router(config)#show ip route

只需告诉路由器该网段怎么走，不需要配置每一个IP,即上图的12.0网段(根据子网掩码位与得出，所以主机部分要写0)
配置好网络之后需要等待一段时间，马上ping可能会丢包，只要之后网络全通就行了
路由表并没有给分组指明到某个网络的完整路径
交换机是根据mac来转发数据的
计算机要跨网段通讯，必须要配网关，除非目标IP地址在本网段，不需要经过路由器
ip helper-address 配置DHCP服务器的地址(不需要子网掩码)，此路由器成为DHCP中继
